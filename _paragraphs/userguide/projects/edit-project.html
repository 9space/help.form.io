Projects can be edited by going to the home page and when logged in and clicking Edit next to the project name. When within a project, you can also get to the edit page by clicking the Edit tab at the top right of the page.

Subdomain

Be careful changing the subdomain as it will change the base path of all integrations with your project. Only do this if you really know what you are doing.

When done, press the Save button
