Project settings are where specific configuration is set for a project. Currently this is only used for setting up the email gateway but this will be expanded to include additional settings in the future.

Project settings can be edited by going to the home page and clicking Settings next to your project. They can also be accessed when within a project by using the Settings tab at the top right of the page.

Email settings

Enter the email credentials for the email provider of your choice and click Save Settings when done.
