New projects can be created from the home page when logged in. Start by clicking the New Project button.

Project Title

Enter the Project title in the Project Title field. This is the administrative name that will be used to refer to your project on form.io.

Subdomain

Enter a unique subdomain for your project. There are certain reserved subdomains that cannot be chosen and each project must have a unique domain name. This will be used for creating a unique API for your project. Choose carefully as this may break things if you change it later.

Description

Enter a description for your project. This is only for your help to remember what a project does.

When done, click Create Project and your new project will be created.
