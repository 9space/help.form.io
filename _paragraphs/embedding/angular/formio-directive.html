Attributes

src - The src attribute is used to pass in the path to the form or form and submission to be displayed.

formAction - To use a custom form action you can specify it as an attribute. This will change where the submit event goes to.

form - Alternatively, you can pass in the form definition as an attribute on the formio directive.

submission - If a submission has already been loaded, it can be passed in as an attribute.

readOnly - Forms can be made read only which can be useful for viewing submission results.

Form.io forms can be embedded direction into your angular application with one line of code. Simply pass in the name of the form to the directive and the Formio module will render out the form for you. If you only pass in the form name, a form with default values will be rendered. This is useful for Creating operations.

<formio src="{formName}"></formio>

You can also render out a form for a previous submission with the values already set in it. This is useful for Editing functionality to allow users to edit previous submissions.

<formio src="{formName}/submission/{submission_id}"></formio>
