Attributes

src - The src attribute is used to pass in the path to the form or form and submission to be displayed.

form - Alternatively, you can pass in the form definition as an attribute on the formio directive.

submission - If a submission has already been loaded, it can be passed in as an attribute.

formAction - To use a custom form action you can specify it as an attribute. This will change where the submit event goes to.

resourceName - The resource name is used in the delete dialogue for confirmation. It can be used to change the default.

The delete directive will show a delete confirmation dialogue and will either cancel or delete based on the selection by the user. Events are triggered for either a cancel or a delete.

<formio-delete src="{formName}/submission/{submission_id}"></formio-delete>
