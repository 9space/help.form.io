In order to embed your Form.io project in your angular application, you will need to add the formio Module. The ngFormio Module will create all scopes, providers and directives needed to embed Formio components within your app.

Bower

To add with bower, add formio to the bower dependencies and run bower install

Add 

<script src="/bower_components/ngFormio/dist/formio.js"></script>
to your index.html page.

NPM

To add with npm, run npm install --save formio

Add 

<script src="/node_modules/ngFormio/dist/formio.js"></script>
to your index.html page.
