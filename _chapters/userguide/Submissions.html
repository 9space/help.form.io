--- title: Submissions book: userguide weight: 100 chapter: submissions ---

Submissions are the values that were submitted to the form you built. These can be thought of as "objects" within the system and operate very much like them. You can get to an administrative view of submissions by editing a form and clicking on the Submissions tab in the top right of the page. You can also embed submissions directly using a <formio> angular directive or you can query the api directly. See Application Embedding and Developer API for more information.
